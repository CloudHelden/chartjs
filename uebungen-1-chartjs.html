Aufgabe 1: Erstes Balkendiagramm

📊 Wann nutzt man Balkendiagramme?

Balkendiagramme sind perfekt um kategorische Daten zu vergleichen. Sie zeigen unterschiedliche Werte für verschiedene Kategorien - wie Verkaufszahlen verschiedener Produkte, Bewertungen von Filmen oder Punkte von Sportteams.

Ziel: Erstelle ein einfaches Balkendiagramm mit diesen Daten:

  • Labels: "Äpfel", "Bananen", "Orangen"
  • Werte: 12, 19, 7
📖 Dokumentation: Bar Chart Documentation
<!-- HTML --> <canvas id="meinChart"></canvas> <!-- JavaScript --> <script> const ctx = document.getElementById('meinChart'); const chart = new Chart(ctx, { type: '???', // Welcher Typ für Balken? data: { labels: ['???', '???', '???'], // Deine Labels hier datasets: [{ label: 'Verkäufe', data: [???, ???, ???] // Deine Zahlen hier }] } }); </script>

💡 Tipp: Schaue in der Dokumentation nach dem "type" für Balkendiagramme!

Aufgabe 2: Tortendiagramm mit Farben

🥧 Wann nutzt man Tortendiagramme?

Tortendiagramme zeigen Anteile eines Ganzen. Sie sind ideal um zu zeigen, wie sich 100% auf verschiedene Kategorien aufteilen - wie Marktanteile, Budgetverteilung oder Umfrageergebnisse in Prozent.

Ziel: Erstelle ein Tortendiagramm für Lieblingsfarben:

  • Rot: 25, Blau: 30, Grün: 20, Gelb: 15
  • Verwende passende Farben für jedes Segment
📖 Dokumentation: Pie & Doughnut Chart Documentation
const chart = new Chart(ctx, { type: '???', // Welcher Typ für Tortendiagramm? data: { labels: ['Rot', 'Blau', '???', '???'], datasets: [{ data: [25, 30, ???, ???], backgroundColor: [ '#ff4444', // Rot '#4444ff', // Blau '???', // Grün - welche Farbe? '???' // Gelb - welche Farbe? ] }] } });

Hier kommt dein Tortendiagramm hin

💡 Tipp: Für Tortendiagramm verwende type: 'pie'. Schaue nach "backgroundColor" in der Doku!

Aufgabe 3: Liniendiagramm für Temperaturen

📈 Wann nutzt man Liniendiagramme?

Liniendiagramme sind perfekt für Daten über Zeit. Sie zeigen Trends und Entwicklungen - wie Temperaturverläufe, Aktienkurse, Webseitenbesucher oder Lernfortschritte über Wochen/Monate.

Ziel: Zeige die Temperatur einer Woche:

  • Tage: Mo, Di, Mi, Do, Fr, Sa, So
  • Temperaturen: 18, 22, 25, 23, 20, 24, 26 (°C)
📖 Dokumentation: Line Chart Documentation
const chart = new Chart(ctx, { type: '???', // Welcher Typ für Linien? data: { labels: ['Mo', 'Di', '???', '???', '???', '???', '???'], datasets: [{ label: 'Temperatur (°C)', data: [18, 22, ???, ???, ???, ???, ???], borderColor: '#ff6b6b', // Farbe der Linie backgroundColor: '#ff6b6b20' // Füllfarbe (optional) }] } });

Aufgabe 4: Doughnut Chart anpassen

🍩 Was ist der Unterschied zwischen Pie und Doughnut?

Doughnut Charts sind wie Tortendiagramme, aber mit einem Loch in der Mitte. Das sieht moderner aus und man kann zusätzliche Informationen in die Mitte schreiben. Beide zeigen Anteile von einem Ganzen.

Ziel: Erstelle ein Doughnut-Diagramm und ändere die Legende:

  • Verwende die gleichen Daten wie bei Aufgabe 2
  • Ändere die Position der Legende nach unten
📖 Dokumentation: Legend Configuration
const chart = new Chart(ctx, { type: 'doughnut', // Doughnut statt pie data: { // ... deine Daten hier ... }, options: { plugins: { legend: { position: '???' // 'top', 'bottom', 'left', 'right' } } } });

Hier kommt dein Doughnut Chart hin

💡 Tipp: Die Legende wird über "options" → "plugins" → "legend" konfiguriert!

Aufgabe 5: Achsen beschriften

📝 Warum sind Beschriftungen wichtig?

Beschriftungen machen Charts verständlicher! Ohne Titel weiß niemand worum es geht, ohne Achsenbeschriftungen weiß niemand was die Zahlen bedeuten. Gute Beschriftungen = professionelle Charts.

Ziel: Erstelle ein Balkendiagramm mit beschrifteten Achsen:

  • Y-Achse: "Verkäufe (Stück)"
  • Titel: "Monatsverkäufe"
  • Daten: Jan: 50, Feb: 75, Mär: 60
📖 Dokumentation: Axes Documentation
📖 Titel: Title Configuration
const chart = new Chart(ctx, { type: 'bar', data: { // ... deine Daten hier ... }, options: { plugins: { title: { display: true, text: '???' // Dein Titel hier } }, scales: { y: { title: { display: true, text: '???' // Y-Achsen Beschriftung } } } } });

Hier kommt dein beschriftetes Diagramm hin

💡 Tipp: Vergiss nicht "display: true" für Titel und Achsenbeschriftungen!

Aufgabe 6: Horizontales Balkendiagramm

↔️ Wann horizontale statt vertikale Balken?

Horizontale Balken sind besser wenn du lange Kategorienamen hast oder wenn du Rankings zeigst. Sie sind auch einfacher zu lesen bei vielen Kategorien. Perfekt für: Umfrageergebnisse, Top-Listen, Skillbewertungen.

Ziel: Erstelle horizontale Balken für Programmierskills:

  • HTML: 90, CSS: 75, JavaScript: 60
  • Verwende horizontale Balken
  • Füge passende Farben hinzu
📖 Dokumentation: Horizontal Bar Chart
const chart = new Chart(ctx, { type: 'bar', // Immer noch 'bar' data: { labels: ['HTML', 'CSS', 'JavaScript'], datasets: [{ label: 'Skill Level (%)', data: [???, ???, ???], backgroundColor: ['#e34c26', '#1572b6', '#f7df1e'] }] }, options: { indexAxis: '???' // 'x' oder 'y' für horizontal? } });

Hier kommt dein horizontales Balkendiagramm hin

💡 Tipp: Für horizontale Balken brauchst du indexAxis: 'y' in den options!
